React Native 500报错指向 '<'

ocsxxi   in 西安，中国
2019-04-24
  • ANDROID
  • JAVASCRIPT
  • REACT
  • REACT NATIVE

# 前提：

配置好了React Native并且按照官网的教程，生成了我的AwesomeProject，并且成功运行。

# 复现

接下来按照官网教程生成 Hello World程序，代码如下

import React, { Component } from 'react';
import { Text, View } from 'react-native';
 
export default class HelloWorldApp extends Component {
  render() {
    return (
        <View style={{ flex: 1, justifyContent: "center", alignItems: "center" }}>
          <Text>Hello, world!</Text>
        </View>
    );
  }
}

将上面的代码覆盖App.js后，在命令行中输入：

react-native run-android

虚拟机反馈报错500，并在cmd中用红色的^指向return中第一个View的开始符号“<”

# 解决

关闭含有server的窗口，重新输入命令：

react-native start

然后将应用部署至虚拟机，输入命令：

react-native run-android

虚拟机在屏幕中央显示了Hello World，成功运行，问题解决。
