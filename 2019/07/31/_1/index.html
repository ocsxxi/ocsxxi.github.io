# 问题描述

因为工作需要，所以自己参考了 https://blog.csdn.net/u012111923/article/details/80705141#comments (opens new window) 大佬的教程，根据教程进行了一次测试。

# 解决方法

是因为JMeter的机制，在断言成功的状态下并不会显示断言是否成功，仅会在断言失败的情况下才会显示错误信息。
接下来，在Response Assertions的界面上，随便打一段匹配文本上去，让它进行匹配，如下图所示
(opens new window)
现在点击运行，我们再来看看断言结果
(opens new window)
